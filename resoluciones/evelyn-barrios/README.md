# Resoluciones de Ejercicios - JavaScript

## Justificación del Ejercicio 01: Variables y constantes - Videojuegos Shooter

La resolución del primer ejercicio se estructuró con el objetivo de aplicar correctamente los fundamentos de declaración de datos en JavaScript. A continuación, se exponen los criterios técnicos utilizados:

- **Uso de `const`**: Se implementó para definir aquellos valores que permanecen inmutables durante el ciclo de vida de la ejecución, tales como el nombre del jugador o el tipo de arma base. Esto garantiza la integridad de los datos que no requieren reasignación.
- **Uso de `let`**: Se reservó para variables cuyo estado es inherentemente dinámico, como los puntos de vida o la cantidad de munición, permitiendo así la actualización de los valores conforme el flujo del programa lo requiere.
- **Trazabilidad**: Se integró el uso de `console.log` para verificar el estado de las variables en la consola, facilitando la depuración y asegurando que la lógica de inicialización fuera coherente con los requisitos del ejercicio.

Este enfoque permite demostrar una comprensión clara de la diferencia entre constantes y variables, así como la importancia de elegir el tipo de declaración adecuado según la naturaleza del dato.
