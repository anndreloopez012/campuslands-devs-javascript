# Resolución Ejercicio 03 - Operadores Matemáticos

**Autor:** Evelyn Barrios  
**Materia:** Programación con JavaScript

## Descripción
Esta entrega corresponde a la implementación de operaciones aritméticas fundamentales en JavaScript, aplicadas a un escenario de estadísticas deportivas (fútbol).

## Justificación Técnica

En la solución se han integrado los cinco operadores aritméticos básicos para procesar la información de un encuentro deportivo:

1.  **Suma (`+`)**: Utilizada para calcular la cifra total de goles marcados en el evento.
2.  **Resta (`-`)**: Aplicada para determinar la brecha o diferencia de anotaciones entre los dos equipos rivales.
3.  **Multiplicación (`*`)**: Empleada para calcular el puntaje obtenido en la tabla de posiciones basándose en el resultado (3 puntos por victoria).
4.  **División (`/`)**: Utilizada para obtener el rendimiento promedio de anotaciones por cada tiempo de 45 minutos.
5.  **Módulo (`%`)**: Implementada para verificar propiedades lógicas del resultado, como determinar si el total de goles es un número par o impar.

## Decisiones de Diseño

*   **Contexto Local**: Se seleccionaron equipos representativos de Guatemala para que el flujo de datos tenga una lógica coherente y familiar.
*   **Código Limpio**: Se evitaron estructuras complejas para priorizar el entendimiento de la sintaxis de los operadores.
*   **Presentación de Datos**: Se utilizaron *Template Literals* y operadores ternarios para asegurar que la información mostrada en consola sea profesional y legible para cualquier usuario.

Este ejercicio valida el dominio de las operaciones matemáticas esenciales, las cuales son la base para algoritmos más avanzados en el desarrollo de software.
